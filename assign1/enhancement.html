Navbar


In this webpage, I have used Bootstrap to make the Navigation Bar more attractive and visibe.If it is done by using raw CSS, then it will take relatively more time. Since Bootstrap has pre-defined design,the navbar is responsive by default.When the webpage is collapsed, the menus are placed in three horizontal lines for using Bootstrap.If anybody clicks there, menus will be seen in vertical position.I have also added a Search Box in the navbar so that it is easy for the users to find their desired location.



Footer

In this webpage,we have hold additional components in Footer like Links,Copyright,Products etc. by using Bootstrap.All the components are shown in column-wise.It looks more colorful,informative and user-friendly.The footer is responsive.That's why all the components are shown in vertical position when the webpage is collapsed.



Color

I have used Bootstrap Colors in this webpage.It is more customized and appealing to users.
